# Sunofriend social media kit

Use these assets and messages to introduce Sunofriend and the worked example
without implying endorsement by any commercial platform or retailer.

> **Before publishing:** commit and push this kit so the public repository links
> expose the artwork, MIDI pack and compatibility form. The draft copy below
> deliberately calls Sunofriend a public project rather than open source because
> the repository does not yet contain a `LICENSE` file. Choose MIT, Apache-2.0,
> GPL or another licence before making an open-source claim or defining reuse
> rights for code, artwork and example MIDI.

## Threads or Instagram

> I built Sunofriend because AI-generated music can sound exciting but still leave muddy, hard-to-edit instruments. It converts separated stems into timing-locked MIDI, keeps different drum timbres, separates bass/keys choices, and labels what was observed, repaired or inferred.
>
> “The Aisle at Lidl” is my worked example using Suno, Moises, Sunofriend and GarageBand. Listen to Version 1: https://soundcloud.com/ezzye-1/the-aisle-at-lidl
>
> The code and example MIDI are open for testing: https://github.com/N9-Developer-Empowerment/sunofriend
>
> I would especially like contributors to try other DAWs, music software, AI generators, stem separators and MIDI hardware—and raise compatibility issues with timing evidence.
>
> Compatibility report: https://github.com/N9-Developer-Empowerment/sunofriend/issues/new?template=daw-ai-compatibility.yml
>
> #AIMusic #MusicProduction #MIDI #MusicTech #GarageBand

## Facebook

> I have been developing Sunofriend, a public music-tech project that turns separated AI-music stems into editable, timing-locked MIDI. The aim is not to replace Suno, Moises or GarageBand, but to connect them: generate, separate, convert, then choose real software instruments and finish the production.
>
> My song “The Aisle at Lidl” is now the public worked example. Version 1 is here: https://soundcloud.com/ezzye-1/the-aisle-at-lidl
>
> The repository includes a lightweight MIDI pack showing two kick timbres, three bass choices, separated keys roles and a reconstructed arrangement: https://github.com/N9-Developer-Empowerment/sunofriend
>
> If you use another DAW, AI music tool, separator or MIDI setup, I would love a compatibility report: https://github.com/N9-Developer-Empowerment/sunofriend/issues/new?template=daw-ai-compatibility.yml

## Posting notes

- Lead with the musical problem and audible example, not a generic “AI app”
  announcement.
- Put the SoundCloud link before the repository link when speaking to music
  listeners; reverse the order for developer/music-tech audiences.
- Invite one concrete action: listen, try the example MIDI in another DAW, or
  file a compatibility report.
- Do not use third-party logos or imply affiliation with Suno, Moises, Apple,
  GarageBand, SoundCloud, Lidl or any social platform.
